Variables

Think of a variable as a box/ container that can hold information. It can hold numbers like (1, 2.5, 35, -107, etc.). Variables can also hold multiple words called strings examples ("hello world", "Kevin") - Example:
function welcome()

Parameters

Parameters are the names/variables given to each argument. Example is
function Name(parameter1, parameter2, parameter3) - Example:
function welcome(name) {

Commands

Commands are the actions the function will run - Example:
alert("Hello " + name);

Arguments

Arguments are the parameters REAL values passed through the function - Example:
welcome("Kevin");


Example:
function welcome(name) {
alert("Hello " + name);
}
welcome("Kevin");

output = Hello Kevin
